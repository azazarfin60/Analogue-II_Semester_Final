### Q.7
* **(a)** Discuss the step-by-step internal working principles and timing cycle of a 555 timer configured as an astable multivibrator. **[03 Marks, CLO3]**
* **(b)** Determine the oscillating frequency and draw the timing capacitor voltage waveform ($v_c(t)$) along with output voltage waveform ($V_o(t)$) for the 555-timer astable circuit shown below. **[04 Marks, CLO2]**

##### Circuit Diagram (Q.7(b)):
```
                +5V
                 |
        +--------+--------+
        |                 |
       [ ] 7.5k           |
        |                 |
        +--------(8)      | (4)
        |  (7)    |       |  |
        |   |   +-----------+
        |   +---|           |
       [ ] 7.5k |    555    |
        |       |   Timer   |
        +-------|           |---(3)---> Output (Vo)
        |  (2,6)|           |
        +---|   +-----------+
        |   |     |       |
        +---+    (1)     (5)
        |         |       |
       === 0.1u   |      === 0.01u
       ---        |      ---
        |         |       |
      --+---------+-------+------- GND
```

##### AI-Ready Structural Walkthrough & Parameter Definitions:
1. **555 Astable Connections:**
   * Resistor $R_A = 7.5\text{ k}\Omega$ (connected between $+5\text{ V}$ and Pin 7).
   * Resistor $R_B = 7.5\text{ k}\Omega$ (connected between Pin 7 and Pins 2/6).
   * Timing capacitor $C = 0.1\ \mu\text{F}$ (connected between Pins 2/6 and Ground).
   * Control voltage bypass: Pin 5 connected to ground via $C_C = 0.01\ \mu\text{F}$.
2. **Frequency Calculations:**
   $$
   t_{high} = 0.693 (R_A + R_B) C = 0.693 \times (7.5\text{ k}\Omega + 7.5\text{ k}\Omega) \times 0.1\ \mu\text{F} \approx 1.04\text{ ms}
   $$
   $$
   t_{low} = 0.693 R_B C = 0.693 \times 7.5\text{ k}\Omega \times 0.1\ \mu\text{F} \approx 0.52\text{ ms}
   $$
   $$
   f_o = \frac{1.44}{(R_A + 2 R_B) C} = \frac{1.44}{(7.5\text{ k}\Omega + 15\text{ k}\Omega) \times 0.1\ \mu\text{F}} \approx 640\text{ Hz}
   $$
