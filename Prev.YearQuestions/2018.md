```
                      Rf = 330k

                    |   |/
                   [ ] 10k
                   [ ]

```

##### AI-Ready Structural Walkthrough & Parameter Definitions:
1. **Operational Amplifier Model:** Ideal op-amp with high gain, virtual short $V_n = V_p$.
2. **Non-inverting Node Potential ($V_p$):** Voltage divider at non-inverting terminal (+) driven by input $V_2 = 2\text{ V}$:
   $$
   V_p = V_2 \left( \frac{10\text{ k}\Omega}{150\text{ k}\Omega + 10\text{ k}\Omega} \right) = 2\text{ V} \times \frac{10}{160} = 0.125\text{ V}
   $$
3. **Superposition Output Formula:**
   $$
   V_o = -\left( \frac{R_f}{R_1} \right) V_1 + \left( 1 + \frac{R_f}{R_1} \right) V_p = -\left( \frac{330\text{ k}\Omega}{10\text{ k}\Omega} \right) V_1 + \left( 1 + \frac{330\text{ k}\Omega}{10\text{ k}\Omega} \right) V_p
   $$
   $$
   V_o = -33 V_1 + 34 V_p
   $$

##### Circuit Diagram (Q.7(b)):
```
                     +10V (Vcc)
                      |

                      |

                      |       |

                     |  +-----+---------------- Pin 6 (Threshold) / Pin 2 (Trigger)
                     |  |

                     | ---
                     |  |
                    GND GND
```

##### AI-Ready Structural Walkthrough & Parameter Definitions:
1. **Diode Impact on Charging/Discharging Path:**
   * **Charging Path (Output High):** The diode is forward-biased, bypassing resistor $R_B$. The timing capacitor $C = 0.1\ \mu\text{F}$ charges directly through $R_A = 1\text{ k}\Omega$ from $\frac{1}{3}V_{CC}$ to $\frac{2}{3}V_{CC}$.
     $$
     t_c \approx 0.693 R_A C
     $$
   * **Discharging Path (Output Low):** The diode is reverse-biased. The timing capacitor $C$ discharges through resistor $R_B = 10\text{ k}\Omega$ into Pin 7 (discharge transistor collector).
     $$
     t_d \approx 0.693 R_B C
     $$
2. **Duty Cycle ($D$):**
   $$
   D = \frac{t_c}{t_c + t_d} = \frac{R_A}{R_A + R_B} = \frac{1\text{ k}\Omega}{1\text{ k}\Omega + 10\text{ k}\Omega} = 9.09\%
   $$
