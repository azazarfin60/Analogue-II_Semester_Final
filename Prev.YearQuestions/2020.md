* **(b)** Derive the expression of output voltage $V_o$ for the logarithmic amplifier circuit shown in Figure 7(b). **[04 Marks]**

##### Figure 7(b) Circuit Diagram:
```
                               Ground
                                 |
                                ===
                                 |
                                 B +---+ C
                        +--------|-|   |------+
                        |          |   |      |
                        |        E +---+      |
                        |          | (NPN)    |
                        |          v          |
                        |          |          |
             R          |        -|           |
       o---[   ]---+----+-------|  \          |
      V_i          |            |   >---------+-----o V_o
                   |            |+/
                  ===           |
                  GND          ===
                               GND
```

##### AI-Ready Structural Walkthrough & Parameter Definitions:
1. **Circuit Topology:** Logarithmic amplifier using a transistor in the feedback loop.
2. **Device Connection:** Matched NPN transistor. Base is connected to Ground ($0\text{ V}$). Emitter is connected to op-amp output node $V_o$. Collector is connected to the virtual ground inverting terminal ($-$).
3. **Current Relation:** Input current $I_{in} = V_i / R$. Collector current:
   $$I_C = I_{ES} \cdot e^{V_{BE}/V_T} \approx I_{in}$$
   Since $V_{BE} = V_B - V_E = 0\text{ V} - V_o = -V_o$, we obtain:
   $$V_o = -V_T \ln\left( \frac{V_i}{I_{ES} R} \right)$$
