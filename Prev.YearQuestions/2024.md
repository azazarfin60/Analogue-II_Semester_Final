* **(c)** Calculate the closed-loop output voltage $V_o$ of the multi-stage operational amplifier circuit shown below. The input voltage is $V_1 = 0.1\text{ V}$ DC. **[03 Marks, CO3]**

##### Circuit Diagram (Q.7(c)):
```
        V1 = 0.1V
        ---+------+-------------------------+
           |                                |
           |        Left Op-Amp             |   Right Op-Amp
           |        +-------\               |     +-------\
           |        |   -    \              +----[ ]-   \
           |   +----+         |             |   20k |     |
           |   |    |   +    /              |       |  +  /------+---> Vo
           +--------+-------/               |   +---+----/       |
           |   |                            |   |   |            |
          [ ] [ ] 400k                      |  [ ] [ ] 100k      |
          20k  |                            |  10k  |            |
           |   +----------------------------+   |   +------------+
          ---+                                  +---|
         GND                                        |
                                                   --- GND
```

##### AI-Ready Structural Walkthrough & Parameter Definitions:
1. **Operational Amplifier Stages:** Two ideal op-amps cascaded.
2. **Left Stage (Non-Inverting Configuration):**
   * Non-inverting (+) terminal receives $V_1 = 0.1\text{ V}$ directly.
   * Feedback loop consists of feedback resistor $R_{f1} = 400\text{ k}\Omega$ and resistor $R_1 = 20\text{ k}\Omega$ connected to Ground.
   * Output voltage of Left Stage:
     $$
     V_{o1} = V_1 \left( 1 + \frac{R_{f1}}{R_1} \right) = 0.1\text{ V} \times \left( 1 + \frac{400\text{ k}\Omega}{20\text{ k}\Omega} \right) = 0.1\text{ V} \times 21 = 2.1\text{ V}
     $$
3. **Right Stage (Inverting Summing Configuration):**
   * Non-inverting (+) terminal is directly grounded.
   * Inverting (-) terminal is connected to:
     * $V_1 = 0.1\text{ V}$ via input resistor $R_{in1} = 20\text{ k}\Omega$.
     * Left stage output $V_{o1} = 2.1\text{ V}$ via input resistor $R_{in2} = 10\text{ k}\Omega$.
     * Feedback resistor $R_{f2} = 100\text{ k}\Omega$.
   * Output voltage of Right Stage:
     $$
     V_o = -\left( \frac{100\text{ k}\Omega}{20\text{ k}\Omega} V_1 + \frac{100\text{ k}\Omega}{10\text{ k}\Omega} V_{o1} \right) = -\left( 5 \times 0.1\text{ V} + 10 \times 2.1\text{ V} \right) = -21.5\text{ V}
     $$
